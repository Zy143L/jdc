# 青龙面板小工具 适用于2.2面板

## 修复远程扫码问题 支持AMD64和ARM64架构

## 配套前端地址
https://github.com/Zy143L/JDC_WEB

## 配套青龙2.2-066 不升级版本
https://hub.docker.com/r/limoe/qinglong

### 使用方法 不再赘述 会用的都会用 

```
nohup ./JDC  1>/dev/null 2>&1 & #AMD64
```

```
nohup ./JDC_arm64  1>/dev/null 2>&1 & #ARM64
```
